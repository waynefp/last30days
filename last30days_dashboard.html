Your Hostinger VPS IP address and port

Set during VPS setup — check /opt/last30days/api/.env

VPS Setup Instructions

  1. 1
    Go to your Hostinger hPanel → VPS → Browser Terminal
  2. 2
    Run this command to download and execute the setup script:
    curl -s https://raw.githubusercontent.com/waynefp/last30days/master/setup_vps.sh | bash
  3. 3
    The script will ask for your OpenAI and xAI API keys, then ask you to set an API key for this dashboard. Note that key down.
  4. 4
    Enter your VPS IP and API key above, click Save, then Test Connection.
  5. 5
    Once connected, go to the Research tab and run your first search!
